# Сложность чтения книги

Это приложение анализирует английскую книгу (.txt), чтобы оценить, насколько она будет сложна для чтения.

### Что делает приложение

- Загружаете текст книги (.txt).
- Программа приводит слова к начальной форме (лемматизация), строит частотный словарь.
- Вы проходите короткий тест (да/нет, знаете ли вы слово).
- Программа определяет вашу границу словарного запаса для этой книги.
- Строится график сложности книги — где и сколько незнакомых слов.
- Можно оценить комфортность чтения (зелёный / жёлтый / красный уровни).
